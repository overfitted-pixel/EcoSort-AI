# ♻️ EcoSort AI

## Intelligent Waste Segregation & Sustainable Recycling Assistant

### Project Overview

EcoSort AI is an AI-powered sustainability assistant that helps users correctly identify waste and provides proper disposal and recycling recommendations.

The project supports **UN Sustainable Development Goal 12 (Responsible Consumption and Production)** by promoting responsible waste management.

## Features

- Waste Classification
- Proper Disposal Guidance
- Environmental Impact Information
- Eco Score
- Sustainability Tips
- User-friendly Interface

## Technologies Used

- Python
- Streamlit
- IBM Granite (Conceptual Integration)
- Prompt Engineering

## SDGs

- SDG 12 – Responsible Consumption and Production
- SDG 11 – Sustainable Cities
- SDG 13 – Climate Action
